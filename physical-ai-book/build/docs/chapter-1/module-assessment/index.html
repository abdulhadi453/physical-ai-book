Module 1 Assessment: Foundations of Physical AI

Complete this comprehensive assessment to validate your understanding of the foundational concepts of Physical AI. Achieve a score of 85% or higher to demonstrate mastery per spec FR9.

Section 1: Physical AI Fundamentals (25 points)​

Question 1​

What distinguishes Physical AI from traditional AI systems?

  • a) Physical AI uses more advanced algorithms
  • b) Physical AI operates directly in physical environments with perception-action loops
  • c) Physical AI requires more computational power
  • d) Physical AI is only used in robotics

Question 2​

Which of the following is NOT a key component of Physical AI systems?

  • a) Sensors
  • b) Actuators
  • c) Control systems
  • d) Virtual reality displays

Question 3​

True/False: Physical AI systems must always include all three components: sensors, actuators, and control systems.

Question 4​

Explain in 2-3 sentences how embodiment influences intelligence in Physical AI systems.

Question 5​

Which of the following best describes the role of embodiment in Physical AI?

  • a) It provides a housing for computational systems
  • b) Physical form fundamentally shapes interactions and capabilities
  • c) It makes robots look more human-like
  • d) It reduces the need for programming

Section 2: Perception-Action Loops (25 points)​

Question 6​

What are the three main components of a perception-action loop?

  • a) Input, Process, Output
  • b) Perception, Reasoning, Action
  • c) Sensors, Computer, Actuators
  • d) Data, Algorithm, Movement

Question 7​

Why is feedback important in perception-action systems?

  • a) It makes the system faster
  • b) It enables adaptation to environmental changes
  • c) It reduces computational requirements
  • d) It eliminates the need for sensors

Question 8​

In a closed-loop control system, what happens when the system receives feedback?

  • a) It stops operating
  • b) It adjusts its behavior based on the feedback
  • c) It switches to open-loop mode
  • d) It ignores the feedback

Question 9​

Describe a practical application where a perception-action loop would be essential, and explain how each component (perception, reasoning, action) would function in your example.

Question 10​

True/False: All Physical AI systems must implement perception-action loops.

Section 3: Mathematical Foundations (25 points)​

Question 11​

What does a homogeneous transformation matrix represent?

  • a) Only rotation
  • b) Only translation
  • c) Both rotation and translation
  • d) Only scaling

Question 12​

What is forward kinematics?

  • a) Calculating joint angles from end-effector position
  • b) Calculating end-effector position from joint angles
  • c) Calculating required forces for movement
  • d) Calculating sensor positions

Question 13​

Why are coordinate systems important in robotics?

  • a) They make programs run faster
  • b) They provide reference frames for spatial reasoning
  • c) They reduce memory usage
  • d) They simplify sensor design

Question 14​

A 2D robot arm has two links of length L1 = 5 units and L2 = 3 units. If both joints are at 90 degrees (π/2 radians), what is the position (x, y) of the end-effector relative to the base? Show your calculations.

Question 15​

Explain how mathematical models enable precise control of Physical AI systems.

Section 4: Application and Integration (25 points)​

Question 16​

How does morphological computation benefit Physical AI systems?

  • a) It makes computation faster
  • b) Physical properties can perform computations that would otherwise require algorithms
  • c) It reduces memory requirements
  • d) It eliminates the need for sensors

Question 17​

What are affordances in the context of embodied AI?

  • a) Robot capabilities
  • b) Possibilities for action offered by environment-body coupling
  • c) Control algorithms
  • d) Sensor limitations

Question 18​

A robot with flexible, compliant legs would be better suited for:

  • a) Precise manipulation tasks
  • b) Rough terrain navigation
  • c) High-speed wheeled locomotion
  • d) Static positioning

Question 19​

Design a simple Physical AI system for a specific task (e.g., cleaning, delivery, assistance). Describe:

  1. The task and environment
  2. The embodiment (physical form) you would choose
  3. The perception-action loop components
  4. How mathematical principles would apply

Question 20​

How would you validate that your Physical AI system is working effectively? Describe at least 3 metrics or tests.

Answer Key and Scoring​

Section 1 Answers:​

  1. b) Physical AI operates directly in physical environments with perception-action loops
  2. d) Virtual reality displays
  3. False - some systems might be purely sensing or actuating
  4. Sample answer: Embodiment shapes intelligence by constraining and enabling interactions with the environment. The physical form affects what the system can perceive and how it can act, fundamentally influencing cognitive processes.
  5. b) Physical form fundamentally shapes interactions and capabilities

Section 2 Answers:​

  1. b) Perception, Reasoning, Action
  2. b) It enables adaptation to environmental changes
  3. b) It adjusts its behavior based on the feedback
  4. Sample answer: Autonomous vacuum cleaner - Perceives dirt and obstacles via sensors, reasons about optimal cleaning path, and acts by moving and cleaning.
  5. True

Section 3 Answers:​

  1. c) Both rotation and translation
  2. b) Calculating end-effector position from joint angles
  3. b) They provide reference frames for spatial reasoning
  4. With both joints at 90°: x = L1cos(90°) + L2cos(90°+90°) = 0 + 3cos(180°) = 0 + 3(-1) = -3; y = L1sin(90°) + L2sin(90°+90°) = 51 + 3sin(180°) = 5 + 0 = 5. So position is (-3, 5).
  5. Sample answer: Mathematical models provide precise representations of system behavior, enabling accurate control, prediction, and optimization of Physical AI systems.

Section 4 Answers:​

  1. b) Physical properties can perform computations that would otherwise require algorithms
  2. b) Possibilities for action offered by environment-body coupling
  3. b) Rough terrain navigation
  4. Sample answer should include task description, appropriate embodiment, perception-action components, and mathematical applications.
  5. Sample answer should include metrics like task completion rate, efficiency, safety, adaptability, etc.

Scoring:​

  • Each multiple choice question: 2 points
  • Each true/false question: 2 points
  • Short answer questions (4, 9, 14, 15, 19, 20): 5 points each
  • Passing score: 85% (85 out of 100 points)
